# **App Name**: Stellenbosch Mafia War Room

## Core Features:

- Command Dashboard Layout: Render the full multi-panel dashboard layout (Header, Left, Center, Right Panels) with placeholders for integration.
- Chart Screenshot Upload & Viewing: Allow users to upload chart screenshots, which replaces previous uploads to conserve storage, and view them within the Chart Analysis Workspace.
- AI-Powered Chart Analysis: Utilize AI to analyze uploaded chart screenshots, providing a tool for pattern recognition and key insights directly within the Chart Analysis Workspace.
- Global Intelligence Panels: Display market news feeds and an economic calendar, providing essential macro intelligence.
- Session-Based Market Context: Show updates relevant to global trading sessions (Asia/London/New York) in the Left Panel for tactical observation.
- Strategy Engine Recommendations: Leverage AI to generate strategy recommendations, incorporating Probability Radar and Trap Detection from market bias analysis based on collected intelligence.

## Style Guidelines:

- Dark color scheme to reflect the 'War Room' concept, promoting focus and a serious analytical environment.
- Primary color: A deep, muted purple (#796F89), providing a sense of depth and strategic thought.
- Background color: A very dark, desaturated purplish-gray (#26232B), creating a subdued and focused canvas.
- Accent color: A vivid yet serious blue (#387BEB), to highlight interactive elements and critical information against the dark background.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for data-dense displays.
- Utilize clean, technical line icons with a slightly 'command center' or military aesthetic, focusing on clarity and rapid comprehension.
- A structured multi-panel dashboard layout (Header, Left, Center, Right) designed for efficient information consumption and distinct functional zones.
- Subtle and efficient transitions for module loading or panel adjustments, ensuring a smooth user experience without visual distraction from analytical tasks.
